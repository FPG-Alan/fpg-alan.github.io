神经网络学习总结

2017-05-01
nerual network

整个四月份在学习神经网络,今天来进行一下归纳总结。

首先我想尝试描述一下，什么是神经网络，它的本质是什么。

从生物角度来说：
类比人脑，计算机看事物跟人是不一样的，但这过程中可以抽象出来一致的点，两者都是通过学习，一步步认知某个东西究竟是什么。
所以重点在于，如何重现这个学习的过程。
人脑拥有大量神经元，之间通过树突和轴突互联，通过神经元细胞模上的蛋白质通道开关，调节膜内外Na，Ka粒子浓度，以形成电压差，进而形成电流传递信号，信号到达轴突终端，触发释放神经递质，引起相连的后续神经元的兴奋或抑制。
这过程中很多细节我并不清楚，大体来说就是后续神经元对某种信号表现出的兴奋或抑制的行为，最终汇总到一起，完成人脑对某种输入信息的反馈，因此人脑的学习过程也就是神经元对某种信号的兴奋还是抑制的调整，至于如何调整，向什么方向调整，我想应该是取决于某次对输入信息的反馈的反馈，换句话来说，这当中应该存在某种负反馈机制，可以让大脑分辨某次反馈是否达到了预期，达到了则加强兴奋，反之则加强抑制。

神经网络，可以看做是上面的过程的一种模拟，重要的是实现某种代价函数，让整个网络可以衡量某次反馈的正确程度，因此才能对应调整每个线路上的权重大小，也就是学习的过程。

因此我认为，神经网络是一个对大脑的模拟，同样实现了类神经元的兴奋/抑制机制。通过构造权重相关的代价函数，并对这个函数实施梯度下降，同样实现了类神经元的负反馈机制，可以自发调整神经网络中的每个权重，最终让神经网络对某类输入形成我们希望的反馈，就像大脑学会认识某种概念。

从数学角度来说：
这个角度上，我现在可以说的还不多，程度还不够。
以分类任务来说，神经网络所做的实际对把输入空间进行某种扭曲，包括平移，缩放，旋转，升维，最终找到一条线(若输入空间线性可分)，或是一个超平面，可以把输入空间一分为二，一旦找到这个分割线/面，神经网络也就可以对后续输入做出预测。

我的学习目标和学习过程(这就是个流水账,主要是想优化学习方式):
目标:了解神经网络,学会使用tensorflow,学习如何进行图片分类,最后掌握如何对图片进行语义分割。
学习过程:
第一周第一天:
先笼统了解了一下几个关键词的相关概念,对比如机器学习,深度学习,神经网络有了一些基础认知。
看了Tensorfow的官方教程，按照教程配置了docker环境，然而发现基本上看不懂。。。

第一周第二天:
决定先放弃Tensorflow,转而补充基础知识,先看了 超智能体
从这本书里看到了更多的基础信息,由此又转到了 colah’s blog,精读了Calculus on Computational Graphs: Backpropagation一文,感觉大受启发,其后又间断的看了colah blog上的其他文章,均有收获。

第一周第三天:
超智能体看到了人工智能章节,决定先放一放,去补充一些数学知识先。。。

第一周第三天到第二周第二天:
回顾线性代数,主要是矩阵运算
回顾高等数学,主要是微积分

第二周第三天:
不知道从什么地方了解到元胞自动机,这一天花费在这个上面,主要了解了初等元胞自动机,还尝试了一下生命游戏,很有意思!

第二周第四天:
继续超智能体, 看完了梯度下降训练法小节, 转到文章介绍的IMPLEMENTING A NEURAL NETWORK FROM SCRATCH IN PYTHON, 按照教程,一步步实现了自己的第一个线性回归神经网络。

第二周第五天到第三周第一天:
在超智能体中到达了神经网络的章节,发现自己依然存在理解困难,于是转到 coursera 参加了他们的机器学习课程,由Andrew Ng主讲，这个是大牛了,讲的课程也很浅显易懂,听完第一周的课程并且都满分通过了考试,感觉自己终于理解线性回归神经网络了。

第三周第二天:
看完了超智能体, 又转回到Tensorflow, 这次果然轻松很多, 按照官网教程, 使用Tensorflow完成了第一个线性回归神经网络, 用于MNIST手写数字识别。
我发现Tensroflow把很多细节都隐藏起来了, 不需要我自己关注如何进行梯度下降, 如何优化算法。
但对Tensorflow的语法还是有很多不明白的地方。

第三周第三天:
开始看卷积神经网络, 在这之前我所接触的都是全连接神经网络, 不适合于图片分类任务(全连接参数太多)。
首先看的是Tensorflow教程, 好的没错, 又看不懂。

第三周第四天:
看了一文读懂卷积神经网络, 大致了解了什么是卷积, 什么是池化, 并在这里3D Visualization of a Convolutional Neural Network建立了对卷积神经网络的直观印象。

第三周第五天到第四周第三天:
看了斯坦福的cs231课程关于卷积网络部分的笔记CS231n Convolutional Neural Networks for Visual Recognition,并在网易云课堂看了课程的录像

第四周第四天:
按照Tensorflow的教程,完成了第一个卷积神经网络,依然用于识别MNIST手写字体。

第四周第五天:
学习了如何保存Tensorflow训练之后的模型,以及如何读取使用这个模型,使用第四天建立的模型进行了2000次训练，得到了准确度94%的神经网络,并用这个神经网络识别了我自己的手写字体。
这里有点需要注意,自己的手写字体需要经过层层转化才能被神经网络识别,具体如下:
1.在纯色背景写字
2.拍照(或扫描),存入电脑
3.使用python pillow库对图片做预处理,这一步由同事帮我完成(感谢素素~)


主要包括亮度对比度调节和锐化,最后反转,这些都是为了提取主要特征,弱化背景噪点。

4.使用这个库JPG-PNG-to-MNIST-NN-Format对图片做进一步处理,主要包括:
裁切图像到MNIST标准(28*28)
把图片转换为二进制文件。

5.在Tensorflow模型内导入这个文件,执行一次神经网络。注意这里不需要初始化变量，也不需要把代价函数和梯度下降引入计算图。

以上就是这一个月来的流水账了,其中有很多时候走了弯路,并且因为基础知识不足,也浪费了不少时间去死磕不懂的地方,后来发现绕个弯去补充一下基础知识会更加的事半功倍,比如在学习卷积网络的时候, 如果能一开始就从cs231n看起会快很多, 也就不会去stackoverflow上问些蠢问题了。。。

以上就是一个月神经网络的学习过程了, 这一个月对我来说确实有点煎熬, 但同时也一直很亢奋, 但现在的进度还很差, 堪堪算是入门吧。。。
接下来我的计划:
1.继续tensorflow教程, 完成cifar-10训练。
2.继续cs231n课程,了解全卷积网络,了解语义分割。
3.看懂Fully Convolutional Networks for Semantic Segmentation
4.掌握这个项目A Kitti Road Segmentation model implemented in tensorflow.
